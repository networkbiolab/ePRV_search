Running the Script:

Basic Command:

./convert_hmm_to_gtf.py -i /path/to/table_HMM.txt -o /path/to/output.gtf

Command with Custom Prefix:

./convert_hmm_to_gtf.py -i /path/to/input.txt -o /path/to/output.gtf -p Slyco_ePRV


Output:

The script will output a GTF file with gene and exon annotations, using the specified prefix for gene IDs.
